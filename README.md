# ClusterDifferentialSpectralEnvelopes
This repository contains the code used for the article:

Giampiero Salvi, Fabio Tesser, Enrico Zovato, Piero Cosi, ["Cluster analysis of differential spectral envelopes on emotional speech"](https://www.isca-archive.org/interspeech_2010/salvi10_interspeech.html), Interspeech 2010.

DOI: 10.21437/Interspeech.2010-121

Check `main.m` for the code.
